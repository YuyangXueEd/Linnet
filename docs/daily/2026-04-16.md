<h4>1. <a href="http://arxiv.org/abs/2604.13495v1">ADP-DiT: Text-Guided Diffusion Transformer for Brain Image Generation in Alzheimer's Disease Progression</a></h4>

**Authors:** Juneyong Lee, Geonwoo Baek, Ikbeom Jang | **Categories:** cs.CV | **Score:** 10.0/10


![The overview of the proposed ADP-DiT framework. The model conditions on multiple text-encoders text embeddings and structured metadata to guide the denoising process in the latent space.](https://arxiv.org/html/2604.13495v1/adpdit.png)

*Figure 1.* The overview of the proposed ADP-DiT framework. The model conditions on multiple text-encoders text embeddings and structured metadata to guide the denoising process in the latent space.



ADP-DiT is a novel Diffusion Transformer designed for generating subject-specific, longitudinal Alzheimer's Disease (AD) brain MRI scans. It achieves clinically interpretable control by encoding follow-up time intervals and multi-domain patient metadata (demographics, diagnosis, neuropsychology) into a natural-language prompt. **Authors:** Bo Liu, Yulong Zou, Jin Hong | **Categories:** cs.CV | **Score:** 10.0/10



CausalDisenSeg addresses robust brain tumor segmentation under missing MRI modalities by employing a Structural Causal Model (SCM) framework. The core method uses causality-guided disentanglement to explicitly separate the anatomical Causal Factor from the modality-dependent Bias Factor using a CVAE with an HSIC constraint. This allows the model to learn robust segmentations by relying only on the causally relevant features, even when some modalities are absent.

<h4>46. <a href="http://arxiv.org/abs/2604.13470v1">Universality of Gaussian-Mixture Reverse Kernels in Conditional Diffusion</a></h4>

**Authors:** Nafiz Ishtiaque, Syed Arefinul Haque, Kazi Ashraful Alam, Fatima Jahara | **Categories:** cs.LG, stat.ML | **Score:** 7.0/10



This paper proves that conditional diffusion models using reverse kernels defined by finite Gaussian mixtures with ReLU networks can approximate target distributions arbitrarily well in context-averaged conditional KL divergence. This is achieved by decomposing the path-space error and showing that each step reduces to a static conditional density approximation problem, solvable via existing Gaussian-mixture theory and ReLU bounds. The core contribution is establishing the universal approximation capability of this specific class of neural reverse kernels under conditional KL divergence.
